                    Search syntax:
                    

                        Use commas to separate multiple terms.

                        All searches are case-insensitive.

                        By default, all visible codepoint properties are searched, but you can use
                        this syntax to search for specific codepoint properties:
                    

                    
                    
U+0030
cp:48
name:Digit three (aliases are also supported)
script:Latin
block:Basic Latin
category:Number, decimal digit

on:sui
